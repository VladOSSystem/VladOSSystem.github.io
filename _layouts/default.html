E2E Testing

We have decided to integrate testing tools: "Puppeteer" to the project, for testing real user scenario with the server response.

Puppeteer is a Node.js library which provides a high-level API to control headless Chrome or Chromium or to interact with the DevTools protocol. It’s maintained by the Chrome DevTools team.

What puppeteer can do?

  • scrape web pages
  • automate form submissions
  • perform any kind of browser automation
  • track page loading performance
  • create server-side rendered versions of SPA
  • make screenshots
  • simulate mobile browsers
  • create automating testing
  • generate PDF from web pages

Puppeteer is very useful for several task types on the web (crawling, testing, debugging, scrapping, automation..Etc)

In our case, we test all UI components with the server response.
